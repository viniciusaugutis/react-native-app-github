Aplicação em React Native para listagem de repositórios e organizações do github por usuário

## Tecnologias

- React Native
- React Navigation
- React Routes
- Axios
- Async Storage
- CodePush
- OneSignal

## Scripts

### `react-native run-android`

### `react-native run-ios`

### `react-native start`

### `react-native start --reset-cache`
